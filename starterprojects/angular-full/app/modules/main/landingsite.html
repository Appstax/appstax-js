What you must do

Set the application key

This template uses the Appstax User Management module and provides user login and user signup out of the box.

For this to work you must set the appKey in the app.js file. You'll find it in the top of the file.

The app key can be found under Settings in the Appstax web user interface. You need to be logged in.

Important files for this template

index.html

Main html file where AngularJS are initialized and started. You'll find all app dependencies here but not much else.

modules/app.js

The main Angular configuration file. Used for setting up routes using the Angular UI Router. If you want to add pages to your app use the following:

  1. Create your html file and optionally your controller.
  2. Create a route in app.js
  3. Access your site using the ui-sref directive.

landingsite.html (this page)

Used for open content (user not logged in). Uses Bootstrap grid for layout. You'll find it here: modules/main/landingsite.html

styles/site.scss

Sass files that compiles to css. Used for styling the landing page. Read more about Sass.

main.html

Used for tha main page after login. You'll find it here: modules/main/main.html

styles/app.scss

Styling of the logged in modules.
